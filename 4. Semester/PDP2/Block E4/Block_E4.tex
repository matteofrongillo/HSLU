\part*{Block E4 - Effective prompting}
\section*{How to write a good prompt (30 min)}
\subsection*{Rules to follow}
\begin{itemize}
    \item Start with the simplest request
    \item Use clear separators between different tasks, such as
        \begin{verbatim}
            ## INTRODUCTION
            ## OUTPUT STYLE
        \end{verbatim}
        \vspace*{-.4cm}
    \item Use clear words to tell the AI what you want, for instance ``Translate'', ``List'', ``Explain''
    \item Be very specific as you add details: ``Explain the difference between renewable and non-renewable energy to a 16-year-old. Use simple words. Include three examples each.''
    \item Go to the point and remove all the unnecessary information.
    \item Do not be generic and specify what you want: ``\sout{Write a few paragraphs}''.\\
        Better: ``Write 2 paragraphs'' or ``Write a 200 word essay''
    \item Instead of saying what not to do, just say what to do.
\end{itemize}

\subsection*{Good prompt example}
\begin{verbatim}
    You are a librarian

    ## RULES
    Before starting generating the output, ask:
    - How old are you?
    - What is your favorite book?
    - How many books per year do you read?
    - What is your favorite book genre?

    ## GOAL
    Suggest a book based on the user's age, favorite book, number of books read the past year,
    and book genre.

    ## OUTPUT
    - Return a list of 5 books
    - Add a 30 word description for each book
    - Ask the user if they want 5 more books suggested
    - Do not ask further questions

    ## CONSTRAINTS
    - Do not suggest the same book
    - Do not suggest books from the same saga, if any
\end{verbatim}

\subsection*{Bad prompt example}
\begin{verbatim}
Suggest some Tolkien-style books.
\end{verbatim}

\section*{Exercise (15 min)}
Write a good prompt in order to have the AI generate a complete Python script for a simple calculator.

\newpage
\subsection*{Possible exercise solution}
\begin{verbatim}
    You are a Python programmer.

    ## GOAL
    Generate a complete Python script for a simple calculator.

    ## TASK
    The program must:
    - Ask the user to enter the first number
    - Ask the user to choose one operation between +, -, *, /
    - Ask the user to enter the second number
    - Calculate and print the result

    ## RULES
    - The code must be complete and directly runnable
    - The code must print only the calculation
    - Use simple Python only
    - Do not include explanations before or after the code
    - Handle division by zero with an error message
    - Handle invalid operations with an error message

    ## OUTPUT STYLE
    - Write the complete `.py` code
\end{verbatim}

\subsection*{Possible output}
\begin{lstlisting}[language=Python]
    first = float(input("Enter the first number: "))
    operation = input("Choose an operation (+, -, *, /): ")
    second = float(input("Enter the second number: "))

    if operation == "+":
        print(first + second)
    elif operation == "-":
        print(first - second)
    elif operation == "*":
        print(first * second)
    elif operation == "/":
        if second == 0:
            print("Error: division by zero")
        else:
            print(first / second)
    else:
        print("Error: invalid operation")
\end{lstlisting}
Save the Python code as\ \ ``calculator.py''

To run it:
\begin{lstlisting}[language=bash]
>   cd path/to/file/
>   python3 calculator.py
\end{lstlisting}
